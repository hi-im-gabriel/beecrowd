# Beecrowd solutions

My Python solutions for problems from [Beecrowd](https://www.beecrowd.com.br/),
previously known as URI Online Judge.

## Repository structure

Solutions are named after their problem ID and grouped into thousand-wide ranges:

```text
1000-1999/
  1000.py
  1001.py
  ...
2000-2999/
  2000.py
  ...
3000-3999/
  3000.py
  ...
```

For example, the solution to problem `1001` is in
[`1000-1999/1001.py`](1000-1999/1001.py).

The code in this repository reflects my own submissions and learning process,
so solutions may favor simplicity or the approach I was studying at the time.
